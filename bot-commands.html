Discord Bot Commands

  • Greeter codes that can be used to substitute server / user data in your messages:
    • <<user>> → username
    • <<disc>> → discriminator
    • <<ping>> → @user
    • <<memc>> → member count
    • Example: "Welcome, <<ping>>!<<user>>#<<disc>> is member <<memc>> on the server!" becomes "Welcome, @czawacki! czawacki#1234 is member 56 on the server!"

  • ~greeter help
    • Explains how to use the greeter subcommands in detail, since they are more complex than the rest of the commands.

  • ~greeter status
    • Displays the current welcome and goodbye messages' information, if present for the server.

  • ~greeter set (join / leave) (#channel) (message: max 1000 characters) (-img (URL max 1000 characters): optional)
    • Adds (or updates) an entry for the guild to send the new message when a user joins/leaves.
    • Example: "~greeter set join #welcome-channel Welcome to the server, <<ping>>! You are member #<<memc>>!"
    • Example: "~greeter set leave #goodbye-channel Goodbye! Thanks for coming by! -img https://media2.giphy.com/media/26u3Zl2MQWJl2hf2g/giphy.gif"

  • ~greeter reset (join / leave)
    • Removes the join/leave message completely. It will no longer send the corresponding message until you set a message again using "~greeter set."
    • Example: "~greeter reset join"

  • ~leaderboard
    • Get top 10 (or top x where x is the number of people who have sent a message) users with the highest chat scores, and also display the invoking user's position on the leaderboard.

  • ~activity rescan
    • (Should be useless most of the time) Checks for any users in the server that are not in the database, and adds them to it.

  • ~activity user (@user)
    • Returns the user's last sign of activity recorded by the bot. (Records user join, reaction, and message)
    • Example: "~activity @czawacki"

  • ~activity list (number)
    • Returns a report of users who have been inactive for (number) days or more.
    • Example: "~activity list 14"

  • ~nick (@user) (new nickname)
    • Nickname the specified user on the server.
    • Example: "~nick @cazwacki Bot Developer"

  • ~kick (@user) (reason: optional)
    • Kicks the user from the server and DMs them, including the reason if provided.
    • Example: "~kick @cazwacki"
    • Example: "~kick @cazwacki Violated rule 3"

  • ~ban (@user) (reason: optional)
    • Bans the user from the server and DMs them, including the reason if provided.
    • Example: "~ban @cazwacki"
    • Example: "~ban @cazwacki Repeatedly broke rule 3"

  • ~purge (number)
    • Removes the (number) most recent messages from the channel the command was invoked in.
    • Example: "~purge 15"

  • ~mv (number) (#channel)
    • Removes the (number) most recent messages from the channel the command was invoked in and pastes them into the channel called in the command.
    • Example: "~mv 25 #gaming"

  • ~cp (number) (#channel)
    • Copies the (number) most recent messages from the channel the command was invoked in and pastes them into the channel called in the command.
    • Example: "~cp 25 #gaming"

  • ~define (word / phrase)
    • Returns the definition of the word or phrase passed in.
    • Example: "~define discord"

  • ~wiki (word / phrase)
    • Returns the Wikipedia entry for the word or phrase passed in.
    • Example: "~wiki discord (software)"

  • ~urban (word / phrase)
    • Returns the first 5 Urban Dictionary entries for the word or phrase passed in.
    • Example: "~urban discord"

  • ~google (word / phrase)
    • Returns the first 5 Google results for the word or phrase passed in.
    • Example: "~google starlink coverage map"

  • ~image (word / phrase)
    • Returns the first 10 images from Google for the word or phrase passed in. They can be navigated by using the left, right, and stop reactions that pop up.
    • Example: "~image color wheel"

  • ~help
    • Returns a URL to this page.
  • ~about (@user)
    • Returns the server join date, nickname (if applicable), and roles associated with the tagged user.
    • Example: "~about @czawacki"

  • ~profile (@user)
    • Returns a 512x512 version of the tagged user's profile picture.
    • Example: "~profile @czawacki"

  • ~perk (perk name)
    • Returns the entry of the perk from the Dead by Daylight Wiki.
    • Example: "~perk We'll Make It"

  • ~shrine
    • Returns the current shrine of 4 perks that can be purchased with shards.

  • ~autoshrine (#channel)
    • Designates the channel for shrine updates from the DBD Twitter account.
    • Example: "~profile @czawacki"
